# Composer Example

Use this prompt when you already have a working draft and want it rewritten safely.

```text
请使用 $academic-paper-composer 继续在我的 working draft 上改稿。
working draft 路径是：[替换成你的 working draft 路径]
学校模板路径是：[替换成你的学校模板或样例路径]

要求：
- 先备份，再改稿。
- 只替换我指定的正文范围。
- 保留原来能证实的图表、数据库说明和运行截图。
- 删除所有模板提示、注释、彩色标题和不实内容。
- 让正文只保留能被真实项目证据支撑的说法。
```

## Expected output

- A cleaned rewrite plan or manuscript section set.
- Clear notes about what was preserved, rewritten, or deleted.
- No unsupported claims, no template noise, no fake screenshots.
